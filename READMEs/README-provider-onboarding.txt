THREDDS Server Providers

1. install THREDDS (https://hub.docker.com/r/unidata/thredds-docker may offer some options)

* need to get a provider to contribute to this section

2. configure THREDDS as ASGS expects (directories, etc)

* need to get a provider to contribute to this section

3. provide designated ASGS Operators access

* create system user
* obtain operator's SSH public key, add it to authorized_host file
